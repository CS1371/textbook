Exercise 3.2 Testing multiple returns

Adapt the original cylinder function as shown in Listing 3.2 and perform the following tests in the Command window:
[a, v] = my_cylinder(1, 1)
a = 12.5664
v = 3.1416
my_cylinder(1, 1)
ans = 12.5664
a = my_cylinder(1, 1)
a = 12.5664
v = my_cylinder(1, 1)
v = 12.5664
So this call:[~, v] = my_cylinder(1, 1); will put the volume in the variable v.
