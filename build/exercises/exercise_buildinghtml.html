Building a Collection

We might want to build a structure array from a cell array with three columns for title, artist and date. The code to do this would be:
% initialize the empty structure array
res = []
for ndx = 1:length(ca)
% create a new structure
str = struct('title', ca{ndx,1}, ...
'artist', ca{ndx,2}, ...
'date', ca{ndx,3});
% append to the emerging result
res = [res str]
end
% nothing to do on exit
